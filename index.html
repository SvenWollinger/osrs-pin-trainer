How do i use this?

1) Use the left inputs to create a new bookmark link

2) Save the link in a password manager for easy access

Why save the link in a password manager?

While the pin in the URL is hashed, its trivial to hash all the possible pins to get yours.

This could happen by having your bookmarks ratted, in the case that you bookmarked it in your browser.

Thats why i recommend to save the link in a safe place, like a password manager.

(The pin is hashed in the url so that someone seeing it cant instantly figure out the pin, you still have to keep the url safe!)

Is this safe to use?

Unless you save the link with your hashed pin in an unsafe location (This includes your browser!), this website is safe.

The site runs locally via javascript only and is hosted on github pages via cloudflare proxy.

You are free however to download the source code and run it locally!

Check "How to run this locally" for a guide!
